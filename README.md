# A Federated, Configurable Protocol for Schedule Exchange

Repo _will_ incude
- `lib` `rust` crate that defines that plugs-in needed dependencies and defines `API`.
- `bin` `rust` crate that works as CLI wrapper of `lib` crate, uses `mlua` crate to make app configurable
- optional front-end written in, intended to support any instance of app from this repo, or something alike
